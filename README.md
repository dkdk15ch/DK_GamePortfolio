# PrideBattle
-UnrealEngineを使用した個人製作ゲームです。

## ダウンロード方法
- プロジェクト		：　画面右の「Code」ボタンから「Download ZIP」で取得できます。
- ビルド済みデータ	： [Releases]からZIPファイルをダウンロードしてください。

## 実行手順
- Unreal Engine（5.61以上）で "BurnWithPride.uproject" (旧プロジェクト名なのでご注意)
- ビルド済みデータの場合は ZIP を解凍、実行ファイルを開いてプレイ可能です。

## 動作環境
- Unreal Engine 5.61
- Windows 10/11
